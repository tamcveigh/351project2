Usage: enter command "java Driver <file>.txt"
where file is the text file containing the 
edges. 

The file containing the edges must have numerical named edges
and have an even count of the edges listed.

When asked for source/destination vertices enter the source vertex,
push enter/return, enter destination vertex, 
and then push enter/return again
